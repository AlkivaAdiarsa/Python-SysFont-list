## How to install
First, You have to have Python installed\
if not, install it here: [https://www.python.org/downloads/](https://www.python.org/downloads/)
<hr/>
<img width="1916" height="1014" alt="image" src="https://github.com/user-attachments/assets/8f4eb6ee-4aed-4789-886d-b2c4687ea0b5" />
then make sure if pygame is installed by running this:

```bash
pip show pygame
```
if not installed, install:
```bash
pip install pygame
```
**if everything is installed, download [main.py](https://github.com/AlkivaAdiarsa/Python-SysFont-list/blob/main/main.py)** ,\
run it by right clicking it, then select open with python
